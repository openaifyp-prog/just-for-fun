# Sudoku Royale

A premium, modern Sudoku experience built with vanilla web technologies.

## Features

- **Modern Fluid UI**: Clean light theme with interactive animations.
- **Smart Generation**: Dynamically generated puzzles with unique solutions.
- **Classic Gameplay**: Mistake tracking, timer, and hints.
- **Responsive Design**: Play on any device.

## How to Play

1. Click/Tap a cell to select it.
2. Use the number pad or your keyboard (1-9) to enter numbers.
3. Complete the grid—each row, column, and 3x3 box must contain numbers 1-9.
4. Careful: 3 mistakes and it's game over!

## Tech Stack

- HTML5
- CSS3 (Custom Variables, Grid, Flexbox)
- JavaScript (ES6 Classes)
