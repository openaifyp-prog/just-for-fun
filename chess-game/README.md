# Chess Royale

A premium, modern Chess experience built with vanilla web technologies.

## Features

- **Elite Aesthetic**: Clean board design with high-contrast pieces and fluid highlights.
- **Full Legal Moves**: Move validation for all pieces (including multi-square moves for Rooks/Queens).
- **Interactive Gameplay**: Highlighted valid moves, capture indicators, and last-move tracking.
- **Move History**: Visual log of all moves played in the session.
- **Controls**: Board reset and multi-step Undo functionality.

## How to Play

1. Click a piece of your color to select it. Valid moves will be highlighted.
2. Click a highlighted circle to move, or a red-outlined square to capture an enemy piece.
3. White moves first.

## Tech Stack

- HTML5
- CSS3 (Grid, Flexbox)
- JavaScript (ES6 Classes)
