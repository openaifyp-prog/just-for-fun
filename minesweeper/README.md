# Minesweeper Elite

A premium, modern Minesweeper experience built with vanilla web technologies.

## Features

- **Elite Aesthetic**: Clean light theme with a focus on usability and polish.
- **Progressive Difficulty**: Easy (8x8), Medium (16x16), and Hard (24x24) modes.
- **Smart Mechanics**: Guaranteed safe first click and recursive revealing.
- **Flagging System**: Support for traditional right-click flagging or a dedicated flag mode.

## How to Play

1. Click a cell to reveal it. Numbers indicate how many mines are adjacent.
2. If you click a mine, the game ends.
3. Right-click (or use Flag Mode) to mark cells where you suspect mines.
4. Clear all non-mine cells to win!

## Tech Stack

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- JavaScript (ES6 Classes)
