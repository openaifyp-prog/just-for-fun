# Aegis Engine

A premium Steampunk Tower Defense game built with vanilla JS.

## Features

- **Steampunk Aesthetic**: Industrial brass, mahogany, and parchment UI.
- **Steam-Powered Towers**: Deploy cannons, harpoons, and valves to defend the core.
- **Mechanical Logic**: Grid-based placement and real-time wave scaling.
- **Visual Juice**: Steam particles, gear-based enemies, and sepia-toned filters.

## Controls

- **Select Tower**: Click an icon in the brass bar (bottom).
- **Place Tower**: Click any white space on the blueprint grid (not on the path).
- **Release Steam**: Engage the next wave of automatons.
