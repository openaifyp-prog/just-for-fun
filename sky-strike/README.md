# Sky Strike Elite

A high-performance, canvas-based vertical jet fighter game built with vanilla JS.

## Features

- **Vertical Combat**: Optimized for classic shoot-'em-up vertical scrolling.
- **Fluid Movement**: Physics-based jet controls with momentum and friction.
- **Vibrant VFX**: Particle system for explosions and engine glow.
- **Cyberpunk HUD**: Glassmorphic UI with real-time score and health tracking.
- **Parallax Skies**: Multi-layered background scrolling for a sense of speed.
- **Dynamic Wave System**: Progressively spawning enemies.

## Controls

- **WASD / Arrow Keys**: Move Jet
- **Spacebar**: Fire Cannons
- **P / Escape**: (Planned) Pause Menu

## Tech

- **Rendering**: HTML5 Canvas API
- **Physics**: Velocity/Friction integration
- **Styling**: Vanilla CSS with Backdrop filters
