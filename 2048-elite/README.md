# 2048 Elite — Premium Puzzle Game

A high-end, glassmorphic reimagining of the classic 2048 puzzle game. Designed with a focus on "expensive" aesthetics, smooth animations, and tactile feedback.

## Features

- **Glassmorphism UI** — Deep obsidian board with semi-transparent, blurred tiles.
- **Metallic Gradients** — Tiles transition from brushed silver to glowing obsidian as they grow.
- **Tactile Feedback** — The board physically tilts in the direction of your move (Keyboard or Swipe).
- **Haptic Visuals** — Satisfying merge pulses and screen-shake feedback.
- **Undo System** — Multi-level undo to help you reach the 2048 tile.
- **Smart Scoring** — Real-time performance HUD with a pop-up addition tracker.

## How to Play

- **Desktop**: Use **Arrow Keys** or **WASD** to slide tiles.
- **Mobile**: **Swipe** in any direction to move.
- **Goal**: Merge tiles with the same number to double them, aiming for the **2048** tile and beyond.

## Technology Stack

- **HTML5** & **Vanilla CSS** (for high-performance glassmorphism/animations).
- **Pure JavaScript** Grid Engine.
- **Inter** & **Outfit** modern typography.

## Run

Simply open `index.html` in any modern browser. Optimized for both desktop and high-resolution mobile devices.
